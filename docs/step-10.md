## SEO summary

### JSON-LD schemas injected
| Page | Schema |
|---|---|
| Homepage | `Organization` |
| Category pages | `BreadcrumbList` (3 levels) |
| Product detail pages | `Product` + `AggregateRating` + `AggregateOffer` (from Step 5) |

### Robots / indexing
- All private routes (`/account/*`, `/checkout/*`, `/cart`, `/auth/*`) have `robots: { index: false }` in their layout metadata — prevents accidental indexing of authenticated pages
- `robots.txt` generated by next-sitemap enforces the same disallow rules

### Sitemap priorities
| Route pattern | Priority | Changefreq |
|---|---|---|
| `/` | 1.0 | daily |
| `/products` | 0.9 | daily |
| `/products/[category]` | 0.85 | weekly |
| `/products/[category]/[product]` | 0.8 | weekly |
| `/search` | 0.6 | monthly |

## Error boundaries
- `app/error.tsx` — catches errors in any route segment, shows "Try Again" (calls `reset()`) + Go Home
- `app/global-error.tsx` — catches root layout failures, must provide own `<html>/<body>` — kept minimal intentionally
- Both log to `console.error`; swap the log statement for `Sentry.captureException(error)` when adding error monitoring
